Soalan Lazim (FAQ)

Bagaimanakah data saya disimpan?

Secara lalai, data disimpan secara automatik di pelayar web peranti anda (offline-safe). Ia tidak akan hilang walaupun anda memuat semula halaman.

Adakah pautan Drive automatik?

Ya, sekiranya pautan API Google Web App telah dimasukkan, sebarang fail harian yang dimuat naik akan disimpan dalam folder Google Drive anda secara automatik.

Bagaimana cara eksport data?

Anda boleh memuat turun fail sandaran harian melalui bahagian "Sandaran & Pemulihan" dalam Tab Tetapan.
